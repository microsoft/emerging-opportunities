    CDISC - Teaming up with Microsoft to Develop Open Source Software

    CDISC is teaming up with Microsoft to develop the CDISC Open Rules Engine (CORE), open-source software that executes machine-readable CDISC Conformance Rules.

    To support and grow a community of open-source software developers, CDISC has initiated the CDISC Open Source Alliance (COSA). Several CDISC member organizations as well as individual developers have already committed to participate in COSA. Microsoft will provide ongoing guidance. Checkout what COSA is...
