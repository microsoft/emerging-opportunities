    Blazor & GraphQL Starter Kit

    We created an open source template for anyone interested in creating a app using Blazor WASM backed by GraphQL!

    Check out our starter kit!

    This kit leverages a .Net library called HotChocolate. HotChocolate a middleware between your Client app and Server app which let you build a GraphQL layer over your existing or new .NET application.!
